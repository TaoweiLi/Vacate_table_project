# Vacate Table [LIVE](https://vacate-table.herokuapp.com/))
Vacate Table is a full-stack OpenTable clone where users can make reservations, write reviews and search, for dining spots across San Francisco.

## Technologies
- React
- Redux
- JavaScript
- Ruby on Rails
- AWS S3
- Google Map API
- PostgreSQL
- Webpack
